# Contributing to AVIS-DATALAKE

This is the **Autonomous Visual Information Seeking** ecosystem. Contributions should maintain the **AIFVS-ARTIFACT** schema.

## Development Guidelines
1.  **Comment Objects**: Include the `AIFVS-ARTIFACT` header for AI readability in all code contributions.
2.  **Versioning**: Respect the `/VERSION 1/` and `/GEMINI/` directory structures established by **cvbgod**.
3.  **Cyborg Engine**: Test any changes to the PHP Data Lake engine for state-drift.

## How to Submit
-   Fork the repository.
-   Create a feature branch (`git checkout -b feature/AVIS-Enhancement`).
-   Submit a Pull Request with a detailed description of the visibility improvement.

*AVIS Law: Visibility is the primary metric.*
