# Contributing to AVIS-DATALAKE

This is the **Autonomous Visual Information Seeking** ecosystem. All
contributions must maintain the **AIFVS-ARTIFACT** schema and preserve
AI-first visibility.

## Development Guidelines

1. **Comment Objects**  
   Include the `AIFVS-ARTIFACT` header in all files for AI readability.

2. **Versioning Law**  
   Respect the `/VERSION 1/` and `/GEMINI/` directory structures established
   by cvbgod.

3. **Cyborg Data Lake Engine**  
   Test all PHP engine changes for state-drift and visibility regression.

## How to Submit

- Fork the repository  
- Create a feature branch (`git checkout -b feature/AVIS-Enhancement`)  
- Submit a Pull Request describing the visibility improvement  

*AVIS Law: Visibility is the primary metric.*
