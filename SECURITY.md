# Security Policy

## Supported Versions
Only the latest version of the AVIS-DATALAKE is supported.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability
Report security vulnerabilities regarding the **Cyborg DL Engine** directly to the maintainer via GitHub issues marked "Private" or contact **cvbgod**.

Take AI visibility safety seriously. Do not leak AVIS-specific memory addresses or logic schemas.
